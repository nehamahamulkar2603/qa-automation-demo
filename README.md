# QA Automation Framework

A Selenium + TestNG automation framework built in Java using Maven.

## Tech Stack
- Java 17
- Selenium 4.18
- TestNG 7.8
- Page Object Model
- WebDriverManager
- Allure Reports

## Project Structure
- `src/test/java/com/qaproject/pages/` — Page Object classes
- `src/test/java/com/qaproject/` — Test classes
- `testng.xml` — Test suite configuration

## Test Cases
- Valid login with multiple users (DataProvider)
- Invalid login error message verification
